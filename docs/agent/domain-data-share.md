# 数据共享领域知识

> 本文档为 Agent 知识路由目标，由 AGENTS.md 的 task-based / path-based / vocabulary-based routing 触发阅读。

## 1. 适用范围

本文档适用于以下场景：
- 修改 `service/data_share/` 目录下的逻辑
- 修改静默访问（SilentAccess）相关行为
- 修改跨应用数据共享控制

## 2. 核心概念

### 2.1 静默访问 / SilentAccess

- 跨应用数据共享机制，不启动提供方进程。
- 访问控制和权限校验必须在入口点执行。

### 2.2 数据共享

- 跨应用数据共享涉及权限隔离。
- 共享行为变更需确认不影响已有共享关系。
