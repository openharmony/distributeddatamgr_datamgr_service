# UDMF 领域知识

> 本文档为 Agent 知识路由目标，由 AGENTS.md 的 task-based / path-based / vocabulary-based routing 触发阅读。

## 1. 适用范围

本文档适用于以下场景：
- 修改 `service/udmf/` 目录下的逻辑
- 修改统一数据管理框架相关行为
- 涉及系统级数据标准的变更

## 2. 核心概念

### 2.1 UDMF

- 统一数据管理框架（Unified Data Management Framework）。
- 系统级数据标准，定义跨应用数据交换的格式和协议。
- 变更需确认不影响已有的数据标准和跨应用交互。
